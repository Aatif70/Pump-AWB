## Introduction

The Petrol Pump Management System is a comprehensive digital solution designed to streamline operations at your petrol pump. This application helps manage employees, track fuel inventory, monitor sales, ensure quality control, and generate detailed reports. The system is divided into two main interfaces: Admin Dashboard and Employee Dashboard, each tailored to the specific needs of these user roles.

## Getting Started

### System Requirements

- **Mobile Device**: Android 6.0+ or iOS 12.0+
- **Internet Connectivity**: Stable internet connection required
- **Storage**: Minimum 100MB free space
- **Permissions**: Camera, Storage, and Location access

### Installation

1. **Download**: Get the app from the provided link or app store
2. **Install**: Follow on-screen instructions to install
3. **Update**: Ensure you're using the latest version for all features

### Login

#### Admin Login
1. Open the application
2. Enter your registered email address
3. Enter your password
4. Enter your SAP number (if provided)
5. Tap "Login"

#### Employee Login
1. Open the application
2. Enter your registered email address
3. Enter your password
4. Enter your SAP number (if provided)
5. Tap "Login"

**Note**: If you've forgotten your password, use the "Forgot Password" option to reset it via email.

## Admin Dashboard

As an administrator, you have access to all features of the system. The Admin Dashboard provides a comprehensive overview of your petrol pump's operations.

### Employee Management

#### Add New Employee
1. Navigate to "Employee Management"
2. Tap "Add New Employee"
3. Fill in required details:
   - Personal information (name, contact, address)
   - Role (Admin, Manager, Attendant)
   - Credentials (email, password)
   - Shift preferences
4. Tap "Save" to create the employee record

#### View and Edit Employees
1. Navigate to "Employee Management"
2. Browse the list of employees
3. Tap on an employee to view details
4. Use the "Edit" button to modify information
5. Use the "Deactivate" button to disable access temporarily
6. Use the "Remove" button to permanently delete an employee

#### Assign Shifts
1. Navigate to "Employee Management" > "Shift Assignment"
2. Select the employee from the list
3. Choose the shift date and time
4. Tap "Assign" to confirm

### Fuel Management

#### Monitor Fuel Tanks
1. Navigate to "Fuel Management" > "Tanks"
2. View current levels, capacity, and fuel type for each tank
3. Tap on a tank to see detailed history

#### Update Fuel Prices
1. Navigate to "Fuel Management" > "Prices"
2. Select the fuel type to update
3. Enter the new price
4. Add effective date and time
5. Tap "Update" to confirm changes

#### Track Fuel Quality
1. Navigate to "Fuel Management" > "Quality"
2. View latest quality check reports
3. Filter by date or fuel type

### Nozzle Management

#### Assign Nozzles
1. Navigate to "Nozzle Management"
2. Select a nozzle from the list
3. Tap "Assign"
4. Choose an employee from the dropdown
5. Set assignment duration
6. Tap "Confirm" to save

#### Monitor Nozzle Status
1. Navigate to "Nozzle Management"
2. View all nozzles with their current status
3. Color indicators show:
   - Green: Active and assigned
   - Yellow: Active but unassigned
   - Red: Inactive or maintenance required

#### View Nozzle Readings
1. Navigate to "Nozzle Management" > "Readings"
2. Select a date range
3. View readings for all nozzles
4. Export data as needed

### Quality Control

## Workflows

### Sales Process

1. **Customer Arrival**
   - Greet the customer
   - Ask for fuel type and quantity

2. **Record Request**
   - Open the app and navigate to "Sales" > "New Transaction"
   - Select fuel type and quantity

3. **Dispense Fuel**
   - Use the assigned nozzle to dispense fuel
   - Ensure accurate quantity

4. **Record Transaction**
   - Enter the dispensed amount
   - Select payment method
   - Process payment

5. **Generate Receipt**
   - Create digital or printed receipt
   - Provide to customer

6. **Update Inventory**
   - The system automatically updates inventory levels
   - Verify readings match physical dispensers

### Quality Check Process

1. **Schedule Check**
   - Admin creates quality check schedule
   - Assigns responsible employees

2. **Perform Inspection**
   - Employee receives notification
   - Collects samples following protocol
   - Conducts tests using appropriate equipment

3. **Record Results**
   - Enter test results in the app
   - Compare with acceptable parameters
   - Document any deviations

4. **Submit Report**
   - Complete the quality check form
   - Add notes and observations
   - Submit for review

5. **Flag Issues**
   - Highlight any parameters outside acceptable range
   - Report urgent concerns immediately
   - Recommend corrective actions

6. **Admin Review**
   - Management reviews quality reports
   - Approves or requests additional testing
   - Initiates corrective measures if needed

### Inventory Management

1. **Monitor Fuel Levels**
   - System tracks real-time inventory levels
   - Alerts when levels reach predefined thresholds

2. **Record Deliveries**
   - Document incoming fuel deliveries
   - Verify quantity and quality
   - Update inventory records

3. **Track Consumption**
   - System calculates daily consumption rates
   - Compares with sales data for reconciliation

4. **Generate Alerts**
   - Receive notifications for low stock
   - Get alerts for unusual consumption patterns

5. **Update Stock Records**
   - Maintain accurate inventory data
   - Reconcile physical and system inventory regularly

### Reporting Cycle

1. **Daily Operations Logging**
   - Employees record all transactions
   - Document any operational issues
   - Note customer feedback

2. **Shift-End Reconciliation**
   - Compare nozzle readings with sales data
   - Reconcile cash and electronic payments
   - Document discrepancies

3. **Daily Reports Generation**
   - System compiles daily sales summary
   - Generates inventory movement report
   - Creates exception reports if needed

4. **Weekly Summaries**
   - Aggregate daily data into weekly reports
   - Highlight trends and patterns
   - Flag areas requiring attention

5. **Monthly Analytics**
   - Generate comprehensive monthly performance data
   - Compare with previous periods
   - Analyze profitability by product

6. **Performance Reviews**
   - Evaluate employee performance
   - Assess operational efficiency
   - Identify improvement opportunities

### Shift Sales Workflow

1. **Shift Assignment**
   - Admin assigns employees to shifts
   - System notifies employees of upcoming shifts
   - Employees confirm shift assignments

2. **Shift Start**
   - Employee logs into the app
   - Records starting nozzle readings
   - Verifies fuel prices and inventory levels

3. **Sales Recording**
   - Employee records each transaction during the shift
   - System tracks sales by fuel type and payment method
   - Real-time updates to inventory levels

4. **Shift End**
   - Employee records ending nozzle readings
   - Reconciles cash and electronic payments
   - Submits final shift sales report

5. **Sales Verification**
   - System compares nozzle readings with recorded sales
   - Flags any discrepancies for review
   - Admin approves or requests corrections

6. **Analytics Generation**
   - System generates shift performance metrics
   - Updates daily and monthly statistics
   - Provides insights for business optimization

## FAQ

**Q: How do I reset my password?**
A: Use the "Forgot Password" link on the login screen and follow the instructions sent to your registered email.

**Q: Can I use the app offline?**
A: Limited functionality is available offline. The app will sync data when internet connectivity is restored.

**Q: How often should quality checks be performed?**
A: Quality checks should be performed daily for each fuel type, and after each new delivery.

**Q: How do I report discrepancies in inventory?**
A: Navigate to "Inventory" > "Issues" and create a new discrepancy report with all relevant details.

**Q: Can multiple employees use the same login?**
A: No, each employee must have their own login credentials for security and accountability purposes.

**Q: How do I update fuel prices?**
A: Only administrators can update fuel prices through the "Fuel Management" > "Prices" section.

**Q: What should I do if a customer disputes a transaction?**
A: Navigate to "Sales" > "Transaction History", locate the transaction, and use the "Dispute" option to document the issue for management review.

**Q: How are shift assignments determined?**
A: Shift assignments are managed by administrators based on operational needs and employee availability.

**Q: How do I view hourly sales patterns?**
A: Navigate to "Reports" > "Sales Analytics" > "Hourly Patterns" and select the date you wish to analyze.

**Q: What should I do if there's a discrepancy between nozzle readings and sales data?**
A: Document the discrepancy in "Sales" > "Issues" with as much detail as possible, including photos of the nozzle readings if available.

**Q: Can I edit a shift sales report after submission?**
A: Only administrators can edit submitted shift sales reports. Contact your manager if corrections are needed.

**Q: How are sales statistics calculated?**
A: Sales statistics are calculated based on all transactions recorded during shifts, including total amount, liters sold, and payment method breakdowns.

**Q: How can I track my performance across different shifts?**
A: Navigate to "Profile" > "Performance" > "Shift Analysis" to view your performance metrics across different shifts and time periods.
